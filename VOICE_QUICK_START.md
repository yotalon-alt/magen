# העוזרת הקולית - התחלה מהירה 🎤

## 3 צעדים פשוטים:

### 1️⃣ אפשר הרשאות מיקרופון
בפעם הראשונה, הדפדפן יבקש הרשאה - **לחץ "Allow"**

### 2️⃣ לחץ על כפתור המיקרופון
הכפתור נמצא **בפינה השמאלית התחתונה** (מעל ה-BottomNavigationBar)
- **לבן** = מוכן לשימוש
- **אדום** = מקשיב עכשיו
- **אפור** = לא זמין (רענן את הדף)

### 3️⃣ דבר בעברית!
**דוגמאות לפקודות:**
```
"מעגל פתוח"
"סנן סריקות"
"הצג סטטיסטיקה לפי מטווחים"
"פתח חומר על שבע עקרונות"
```

## פקודות מהירות לפי דף:

| דף | פקודה לדוגמה |
|---|---|
| 📋 **תרגילים** | "מעגל פתוח" / "סריקות" / "מיונים" |
| 📝 **משובים** | "סנן מעגל פרוץ" / "הצג כללי" |
| 📊 **סטטיסטיקה** | "תיקיית מטווחים" / "תרגיל סריקות" |
| 📚 **חומר עיוני** | "פתח מעגל פתוח" / "שבע עקרונות" |

## לא עובד? 🔧

### הכפתור אפור:
1. רענן את הדף (F5)
2. בדוק הרשאות מיקרופון
3. נסה דפדפן Chrome

### לא מזהה:
1. דבר ברור ובקול רם
2. השתמש בפקודות פשוטות
3. בדוק שהמיקרופון עובד

### הודעת שגיאה:
- פתח Console (F12) וחפש את הסיבה
- ראה [VOICE_ASSISTANT_GUIDE.md](VOICE_ASSISTANT_GUIDE.md) למדריך מלא

## 💡 טיפים:
- דבר **לאט** ו**בבירור**
- השתמש ב**מילות מפתח** מהטבלה למעלה
- תופיע **הודעה ירוקה** עם מה שזוהה
- אם לא עובד - נסה **ניסוח אחר**
